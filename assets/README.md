# Assets

Drop your files here. The site expects:

- `profile.jpg` (or `profile.png`) — your profile photo, ideally square
  (e.g. 400×400). The page tries `profile.jpg` first and falls back to
  `profile.png` automatically.
- `cv.pdf` — your CV / resume, linked from the "CV / Resume" section.

These files are not committed yet, so add them before publishing.
